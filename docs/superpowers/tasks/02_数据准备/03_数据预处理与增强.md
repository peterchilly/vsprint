# 数据预处理与增强

> **所属阶段：** 阶段二：数据准备
> **阶段目标：** 构建高质量、规范化的训练/验证/测试音频数据集，用于说话人识别任务。
> **阶段交付物：**
- 划分好的训练/验证/测试音频数据集
- 数据增强策略确定
- DataLoader 实现完成
- 数据质量检查报告

---

## 目标
实现音频数据的预处理和增强策略，提升说话人识别模型的泛化能力。

## 任务清单

### 1. 基础处理
- [ ] 音频重采样到统一采样率（推荐 16kHz，说话人识别标准）
- [ ] 转换为单声道（Mono）
- [ ] 预加重（Pre-emphasis）：`y[n] = x[n] - 0.97 * x[n-1]`
- [ ] 提取声学特征：**FBank（Filter Bank）** 或 **MFCC**（推荐 80 维 FBank）
  - 帧长 25ms，帧移 10ms
  - 加 Hamming 窗
  - 对数能量

### 2. 训练集增强（提升鲁棒性）
- [ ] 加性噪声：在音频中叠加 RIR（房间脉冲响应）和噪声（使用 MUSAN 噪声库）
- [ ] 速度扰动：0.9x / 1.0x / 1.1x 变速
- [ ] SpecAugment：对频谱图进行时间遮蔽和频率遮蔽
- [ ] 随机增益：音量扰动（±5dB）
- [ ] 时长截断/填充：统一输入长度（如 2 秒，16000 采样点 × 2 = 32000）

### 3. 验证/测试集处理
- [ ] 仅做基础处理（重采样 → 单声道 → FBank 提取）
- [ ] 不做随机增强，确保评估稳定性
- [ ] 可选：CMVN（Cepstral Mean and Variance Normalization）归一化

## 验收标准
- [ ] 训练/验证 Feature Extractor pipeline 完成
- [ ] 增强效果可视化验证（波形图/频谱图对比）
- [ ] 增强参数可通过配置文件调整
- [ ] FBank 特征维度正确（如 80 维 × 帧数）
