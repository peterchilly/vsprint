# 数据集选择与获取

> **所属阶段：** 阶段二：数据准备
> **阶段目标：** 构建高质量、规范化的训练/验证/测试音频数据集，用于说话人识别任务。
> **阶段交付物：**
- 划分好的训练/验证/测试音频数据集
- 数据增强策略确定
- DataLoader 实现完成
- 数据质量检查报告

---

## 目标
确定说话人识别任务类型，选择并下载对应的音频数据集。优先使用 VoxCeleb 数据集。

## 任务清单
- [ ] 明确任务类型：说话人识别（Speaker Verification / Identification）
- [ ] 选择数据集：**VoxCeleb1**（首选，1,251 说话人，~15 万条语音）或 **VoxCeleb2**（6,112 说话人，~100 万条语音）
- [ ] 从官方源或 Hugging Face Datasets 下载数据集
- [ ] 验证数据完整性：文件数量、音频格式（.wav/.ogg/.m4a）、采样率一致性、说话人标签一致性
- [ ] 数据存放在 `data/voxceleb/` 目录，创建数据说明文档 `data/README.md`

## 验收标准
- [ ] VoxCeleb 数据集下载完成（至少 VoxCeleb1 dev set）
- [ ] 音频文件数量和格式验证通过
- [ ] 数据目录结构清晰，包含说话人 ID 子目录
- [ ] 数据说明文档已创建（记录样本数、说话人数、音频格式、采样率等）

## 参考数据集
除了 VoxCeleb 之外，在声纹识别（Speaker Verification）、说话人日志（Speaker Diarization）以及多模态视听（Audio-Visual）领域，还有几个非常经典且常作为 Baseline（基准）的开源数据集。

它们可以根据野外环境（In the Wild）、多方言/跨设备（Multi-condition）和多模态（Audio-Visual）等不同的研究侧重点来进行分类：

1. 规模更大、更偏向“普通人”：VoxBlink / VoxBlink2
这是近年学术界为了打破 VoxCeleb 局限于“名人/公开采访”而推出的超大规模数据集（同样源自 YouTube）。

核心特色： 它的数据来源于普通用户录制的短视频、Vlog、生活直播、播客等。相比于字正腔圆的名人采访，VoxBlink 的场景更贴近现实生活中的语音交互，口音和噪声也更加琐碎。

数据规模： 升级后的 VoxBlink2 拥有超过 11 万个说话人（Speakers）、约 1000 万段语音切片，规模远超 VoxCeleb2。目前很多顶尖的声纹模型（如 ECAPA-TDNN 变体）会结合它和 VoxCeleb 混合训练，能把 VoxCeleb1-O 测试集的 EER 刷到极低的水平（接近 0.17%）。

2. 阿里开源的大型多条件数据集：3D-Speaker
由阿里巴巴达摩院开源，专门用于解决真实世界中“跨器件（Device）”、“跨距离（Distance）”和“跨方言（Dialect）”的声纹识别难题。

核心特色： 包含超过 10,000 个说话人。它的特殊之处在于控制变量录制：同一个说话人，会同时用智能手机、即时通讯工具、高保真麦克风在不同的距离（近场、远场）进行录制，同时覆盖了多种汉语方言。

应用场景： 非常适合用来训练和测试智能家居终端（如音箱、车载系统）的声纹锁，能很好地检验模型在“用户换了个手机说话”或“站在 3 米外说话”时的鲁棒性。

3. 多模态视听同步经典：CN-Celeb / CN-Celeb2
由于 VoxCeleb 绝大多数是英文/欧美面孔，清华大学团队推出了针对中文（Mandarin）的类似数据集。

核心特色： 完全对标 VoxCeleb 的自动化流水线，从爱优腾、B站等国内视频平台爬取明星、up主、学者、主播的音视频。

数据规模： 包含 3,000 个左右的中国名人，超过 50 万段语音。包含了大量的电话采访、娱乐节目、演讲、戏剧等极度嘈杂的实际场景。它是研究中文声纹识别和中文多模态视听融合的必选数据集。

4. 经典野外声纹评测集：SITW (Speakers in the Wild)
由 SRI International 团队构建，是一个比较早但非常严谨的“野外”声纹验证数据集。

核心特色： 它的样本不是实验室录制的，而是从公开渠道收集的真实人声（涵盖各类视频、纪录片、露天演讲）。

应用场景： 数据量相比 VoxCeleb 不大（约 299 个说话人），但它的 SITW Eval Set 是学术界非常公认的声纹验证测试集。很多人在 VoxCeleb 上训练完模型后，会拿到 SITW 上去跑测试，看模型的泛化能力（Generalization）到底怎么样。

5. 跨情感声纹研究：ESD (Emotional Speech Dataset)
如果你的语音系统或者 ASR 终端需要处理用户在愤怒、悲伤、兴奋等极端情绪下的声音，VoxCeleb 的不确定性就太大了（它的情绪是随机的）。这时候就需要用到 ESD。

核心特色： 包含 10 个英文和 10 个中文说话人，每个人在严格控制的 5 种情绪状态（纯中性、喜悦、愤怒、悲伤、惊奇）下录制相同文本的短语。

应用场景： 常用于跨情感声纹验证（Enrolled in Neutral, Tested in Anger），或者用于情感语音合成（TTS）和语音转换（VC）。
