# 数据探索与分析（EDA）

> **所属阶段：** 阶段二：数据准备
> **阶段目标：** 构建高质量、规范化的训练/验证/测试音频数据集，用于说话人识别任务。
> **阶段交付物：**
- 划分好的训练/验证/测试音频数据集
- 数据增强策略确定
- DataLoader 实现完成
- 数据质量检查报告

---

## 目标
对 VoxCeleb 音频数据集进行探索性分析，了解音频特征和潜在问题。

## 任务清单
- [ ] 统计说话人（类别）分布，检查每个说话人的语音片段数量
- [ ] 分析音频时长分布（最短/最长/平均/中位数）
- [ ] 分析采样率、声道数、音频格式一致性
- [ ] 可视化部分样本的波形图和频谱图（Waveform + Spectrogram）
- [ ] 检查是否存在损坏文件、静音片段、噪声异常等问题
- [ ] 记录数据集元信息（说话人数、总样本数、平均时长、采样率等）

## 验收标准
- [ ] EDA 报告完成（含可视化图表）
- [ ] 说话人分布不平衡问题已识别
- [ ] 数据集元信息记录完整
- [ ] 数据质量问题已标记
