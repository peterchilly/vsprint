# 数据集划分

> **所属阶段：** 阶段二：数据准备
> **阶段目标：** 构建高质量、规范化的训练/验证/测试音频数据集，用于说话人识别任务。
> **阶段交付物：**
- 划分好的训练/验证/测试音频数据集
- 数据增强策略确定
- DataLoader 实现完成
- 数据质量检查报告

---

## 目标
将 VoxCeleb 数据集划分为训练集、验证集和测试集。

## 任务清单
- [ ] 使用 VoxCeleb 官方划分（推荐）：
  - **VoxCeleb1 Dev** → 训练集（1,211 说话人）
  - **VoxCeleb1 Test (VoxCeleb-O/H/C)** → 测试集（40 说话人，含 trials 文件）
  - 或从 Dev 中划分出一部分作为验证集
- [ ] 确保每个说话人的语音片段在训练/验证中不重叠
- [ ] 保存划分索引（说话人列表 + 文件列表），设置随机种子确保可复现
- [ ] 记录各子集的说话人数、样本数、平均时长
- [ ] 生成 trials 文件格式的评估列表（enrollment / test 对）

## 验收标准
- [ ] 训练/验证/测试集划分完成
- [ ] 说话人身份在训练集和测试集之间不重叠
- [ ] 划分索引文件已保存（JSON/CSV 格式）
- [ ] 各子集统计信息记录完整
