# Python 环境配置

> **所属阶段：** 阶段一：环境搭建
> **阶段目标：** 搭建稳定、可复现的深度学习开发环境，确保后续所有阶段能顺利运行。
> **阶段交付物：**
- 可运行的 Python 虚拟环境
- GPU 验证通过
- 项目目录结构搭建完成
- 实验追踪系统就绪

---

## 目标
安装并配置 Python 虚拟环境，锁定所有依赖版本。

## 任务清单

### 1. Python 安装
- [ ] 安装 **Python 3.9–3.11**（推荐 **3.10**）
- [ ] 验证安装：`python --version`

### 2. 虚拟环境创建
- [ ] 选择 conda 或 venv
- [ ] 创建虚拟环境并激活
- [ ] 验证环境隔离正常

### 3. 依赖锁定
- [ ] 生成 `requirements.txt` 或 `environment.yml`
- [ ] 记录所有依赖的精确版本号

## 验收标准
- [ ] 虚拟环境可正常激活和退出
- [ ] `requirements.txt` 文件存在
