## 5. 训练设备建议

基于当前硬件条件：

| 场景 | 建议 |
|------|------|
| 原型开发 / 小规模实验 | ✅ 本地 GPU 足够 |
| 大规模训练（ImageNet 级别） | ⚠️ 可能需要云服务器或多卡 |
| 生产部署推理 | ✅ 单卡推理性能充足 |

**推荐配置（本地训练）：**
- `batch_size`: 64（224×224 输入）
- `num_workers`: 4-8（取决于 CPU 核心数）
- `amp`: True（混合精度，减少显存占用）
