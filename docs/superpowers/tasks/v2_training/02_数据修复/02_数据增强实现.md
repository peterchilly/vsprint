### Step 1: 创建增强模块 `src/datasets/augmentation.py`

实现以下增强类（作用于 FBank 特征，非原始音频）：

```python
class SpecAugment:
    """SpecAugment: 时间遮蔽 + 频率遮蔽"""
    def __init__(self, time_mask=10, freq_mask=20, num_masks=2):
        ...
    def __call__(self, fbank):
        # fbank shape: (n_mels, n_frames)
        # 随机选择时间区域置零
        # 随机选择频率区域置零
        return fbank

class RandomGain:
    """随机增益：对 FBank 特征加偏移"""
    def __init__(self, gain_range=5.0):
        # gain_range: ±5dB，转为 FBank 域的偏移量
        ...
    def __call__(self, fbank):
        # 随机选择增益值，加到整个特征上
        return fbank

class Compose:
    """组合多个增强"""
    def __init__(self, transforms):
        ...
    def __call__(self, fbank):
        for t in transforms:
            fbank = t(fbank)
        return fbank
```

注意：SpeedPerturbation 需要对原始音频操作，而当前管线用预提取特征，因此暂不实现。SpecAugment 和 RandomGain 可直接作用于 FBank 特征。

### Step 2: 编写测试 `tests/test_augmentation.py`
```python
def test_specaugment():
    """SpecAugment 不改变 shape"""
    fbank = np.random.randn(80, 200).astype(np.float32)
    aug = SpecAugment(time_mask=10, freq_mask=20, num_masks=2)
    out = aug(fbank)
    assert out.shape == fbank.shape

def test_random_gain():
    """RandomGain 不改变 shape"""
    fbank = np.random.randn(80, 200).astype(np.float32)
    aug = RandomGain(gain_range=5.0)
    out = aug(fbank)
    assert out.shape == fbank.shape

def test_compose():
    """Compose 组合增强"""
    fbank = np.random.randn(80, 200).astype(np.float32)
    aug = Compose([SpecAugment(), RandomGain()])
    out = aug(fbank)
    assert out.shape == fbank.shape
```
