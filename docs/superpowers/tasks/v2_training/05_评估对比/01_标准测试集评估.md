# Task 5.1: 标准测试集评估

**阶段**：05_评估对比
**依赖**：T4.2
**预计时间**：1 小时

## 目标
在 VoxCeleb1 标准测试集（40 说话人，4791 条音频）上评估 v2 模型，产出完整评估报告。

## 需要修改/创建的文件
- 创建：`reports_v2/evaluation_report.md`
- 创建：`reports_v2/evaluation_data.json`
- 创建：`reports_v2/roc_curve.json`
- 创建：`reports_v2/det_curve.json`

## 步骤

### Step 1: 准备测试集
- 使用 `data/voxceleb/test/` 目录下的 40 说话人音频
- 为每个说话人提取 FBank 特征（使用与训练相同的参数）
- 生成评估用的 trials 列表（正样本对 + 负样本对）

### Step 2: 执行评估
```powershell
cd C:\Users\Administrator\vsprint
python scripts/evaluate.py --config configs/train_v2.yaml --checkpoint checkpoints_v2/best_model.pth --output-dir reports_v2
```

### Step 3: 收集指标
评估报告必须包含：
- **EER** + 阈值
- **minDCF** + 阈值
- **pos_mean** ± std（同一人余弦相似度）
- **neg_mean** ± std（不同人余弦相似度）
- **ROC 曲线**数据
- **DET 曲线**数据
- **混淆分析**（Top-10 最易混淆说话人对）
- **效率指标**（参数量、FLOPs、推理延迟）

### Step 4: 生成报告
`reports_v2/evaluation_report.md` 应包含：
1. 评估概览（EER/minDCF/pos_mean/neg_mean 表格）
2. 效率指标
3. 混淆分析
4. 错误分析
5. ROC/DET 曲线数据引用

## 产出物
- `reports_v2/evaluation_report.md` — 评估报告
- `reports_v2/evaluation_data.json` — 完整评估数据
- `reports_v2/roc_curve.json` — ROC 曲线
- `reports_v2/det_curve.json` — DET 曲线

## 验证方法
```powershell
python -c "
import json
d = json.load(open('reports_v2/evaluation_data.json'))
m = d['metrics']
print(f'EER: {m[\"eer\"]:.4f}')
print(f'pos_mean: {m[\"pos_mean\"]:.4f}')
print(f'neg_mean: {m[\"neg_mean\"]:.4f}')
assert m['eer'] < 0.10, 'EER not meeting target'
assert m['pos_mean'] > 0.50, 'pos_mean too low'
print('PASS')
"
```
