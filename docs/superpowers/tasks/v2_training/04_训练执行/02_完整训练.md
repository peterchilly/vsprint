# Task 4.2: 完整训练

**阶段**：04_训练执行
**依赖**：T4.1（冒烟测试通过）
**预计时间**：12-24 小时

## 目标
用 v2 配置执行完整训练（50 epoch 或 early stop），产出最优 checkpoint。

## 需要修改/创建的文件
- 产出：`checkpoints_v2/best_model.pth`
- 产出：`checkpoints_v2/last_model.pth`
- 产出：`logs_v2/training.log`

## 步骤

### Step 1: 启动完整训练
```powershell
cd C:\Users\Administrator\vsprint
python train.py --config configs/train_v2.yaml
```
配置中已设 epochs=50, early_stopping_patience=15。

### Step 2: 训练中检查点
每 5 epoch 检查一次：
- EER 是否在下降或稳定（不退化）
- train_acc 是否在上升
- pos_mean 是否在上升
- loss 是否正常收敛

如发现 EER 连续 5 epoch 上升，应暂停训练并排查。

### Step 3: 训练完成验证
训练结束后检查：
- best_model.pth 的 epoch 和 EER
- training.log 是否完整
- checkpoints_v2/ 下是否有定期 checkpoint

### Step 4: 记录最终结果
从 training.log 提取：
- best epoch, best EER
- 最终 pos_mean, neg_mean
- train_acc 最终值

## 产出物
- `checkpoints_v2/best_model.pth` — 最优模型
- `checkpoints_v2/last_model.pth` — 最后一个 epoch 模型
- `logs_v2/training.log` — 完整训练日志

## 验证方法
```powershell
python -c "
import torch
c = torch.load('checkpoints_v2/best_model.pth', map_location='cpu', weights_only=False)
print(f'epoch: {c[\"epoch\"]}')
print(f'best_val_score (EER): {c[\"best_val_score\"]}')
assert c['best_val_score'] < 0.10, 'EER not meeting target'
print('PASS')
"
```

- [ ] best_model.pth 存在
- [ ] training.log 完整记录所有 epoch
- [ ] best EER < 0.10
- [ ] EER 趋势不退化（最后 10 epoch EER ≤ 前 10 epoch）
