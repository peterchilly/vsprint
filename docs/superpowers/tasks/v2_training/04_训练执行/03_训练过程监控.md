# Task 4.3: 训练过程监控

**阶段**：04_训练执行
**依赖**：T4.2
**预计时间**：30 分钟

## 目标
从训练日志中提取逐 epoch 指标，确认训练曲线健康，为后续评估提供数据。

## 需要修改/创建的文件
- 创建：`logs_v2/metrics_per_epoch.json`

## 步骤

### Step 1: 解析 training.log
提取每个 epoch 的：
- train_loss, train_acc
- val EER, minDCF
- pos_mean, neg_mean, pos_std, neg_std
- learning rate

### Step 2: 趋势分析
计算以下趋势指标：
- EER 变化：首 epoch → 末 epoch 的变化量
- EER 是否单调下降（允许小波动）
- pos_mean 是否上升
- train_acc 是否上升
- loss 是否正常收敛

### Step 3: 生成 JSON 报告
```json
{
  "epochs": [
    {"epoch": 0, "eer": 0.20, "loss": 5.0, "acc": 0.05, "pos_mean": 0.20, "neg_mean": 0.01, "lr": 0.0002},
    ...
  ],
  "trend_analysis": {
    "eer_trend": "decreasing",
    "eer_delta": -0.10,
    "pos_mean_trend": "increasing",
    "pos_mean_delta": 0.25,
    "acc_trend": "increasing",
    "loss_trend": "decreasing",
    "healthy": true
  },
  "best_epoch": 35,
  "best_eer": 0.065,
  "final_pos_mean": 0.55,
  "final_neg_mean": 0.005
}
```

## 产出物
`logs_v2/metrics_per_epoch.json`

## 验证方法
```powershell
python -c "
import json
d = json.load(open('logs_v2/metrics_per_epoch.json'))
print(f'epochs: {len(d[\"epochs\"])}')
print(f'best_eer: {d[\"best_eer\"]}')
print(f'healthy: {d[\"trend_analysis\"][\"healthy\"]}')
assert d['trend_analysis']['healthy'] == True
print('PASS')
"
```

## 通过标准
- [ ] JSON 文件存在且包含所有 epoch 数据
- [ ] EER 趋势为 decreasing
- [ ] pos_mean 趋势为 increasing
- [ ] trend_analysis.healthy = true
- [ ] best_eer < 0.10
