# Task 4.1: 冒烟测试

**阶段**：04_训练执行
**依赖**：T2.2, T3.1, T3.3
**预计时间**：1 小时

## 目标
用 v2 配置训练 3 个 epoch，验证模型不坍塌、训练指标健康。

## 需要修改/创建的文件
- 创建：`reports/smoke_test_3epoch.json`

## 步骤

### Step 1: 执行 3 epoch 训练
```powershell
cd C:\Users\Administrator\vsprint
python train.py --config configs/train_v2.yaml --epochs 3
```

### Step 2: 收集指标
从训练输出中提取每个 epoch 的：
- train_loss, train_acc
- val EER, minDCF
- pos_mean, neg_mean

### Step 3: 坍塌检查
判定标准：
- **通过**：EER < 0.20（至少不比随机差太多）
- **通过**：train_acc > 5%（远高于随机 0.083%）
- **通过**：loss 呈下降趋势，但不是骤降到极低值
- **通过**：pos_mean > 0.10（同一人相似度不是接近 0）

### Step 4: 生成 JSON 报告
```json
{
  "epochs": [
    {"epoch": 0, "train_loss": 5.2, "train_acc": 0.08, "eer": 0.18, "pos_mean": 0.25, "neg_mean": 0.01},
    {"epoch": 1, ...},
    {"epoch": 2, ...}
  ],
  "collapse_check": {
    "eer_ok": true,
    "acc_ok": true,
    "loss_ok": true,
    "pos_mean_ok": true
  },
  "verdict": "pass" | "fail",
  "notes": "..."
}
```

## 产出物
`reports/smoke_test_3epoch.json`

## 验证方法
```powershell
python -c "import json; d=json.load(open('reports/smoke_test_3epoch.json')); print(d['verdict'], d['collapse_check'])"
# 预期输出：pass 和所有 True
```

## 通过标准
- [ ] 3 epoch 训练完成无报错
- [ ] EER < 0.20
- [ ] train_acc > 5%
- [ ] loss 正常下降（非坍塌到 0.001）
- [ ] pos_mean > 0.10
- [ ] verdict = "pass"
